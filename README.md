# bioinformatics-practice
A collection of daily beginner Python scripts practicing sequence cleaning, mutation tracking and pipeline design.

### Week 1: DNA & RNA Text Processing
* **Day 1: DNA Sequence Cleaner** (`dna_sequence_cleaner.py`) - Cleans messy database text strings by eliminating whitespace and handling mixed cases.
* **Day 2: Transcription Simulator** (`dna_transcription.py`) - Replicates cellular transcription by converting Thymine (T) bases to Uracil (U).
* **Day 3: GC Content Calculator** (`gc_calculator.py`) - Utilizes custom mathematical operations to predict sequence stability based on G/C ratios.
* **Day 4: Reverse Complement Generator** (`reverse_complement_generator.py`) - Generates the matching double-helix strand using matrix mapping and reverse slicing.
* **Day 5: Motif Finder** (`motif_finder.py`) - Implements an index-shifted `while` loop to catch target sequence patterns.
* **Day 6: FASTA File Parser** (`FASTA_parser.py`) - Manages File I/O to safely parse and sanitize standard raw biological database assets.
* **Day 7: Restriction Site Mapper** (`restriction_site_mapper.py`) - Applies conditional logic blocks to isolate specific EcoRI enzyme target zones.

### Week 2: Proteins, Mutations, & Data Engineering
* **Day 8: RNA Translator** (`RNA_to_protein_translator.py`) - Employs a Python dictionary key-value map to translate single codons.
* **Day 9: Point Mutation Detector** (`point_mutation_detector.py`) - Computes the Hamming Distance across multiple aligned strands via parallel indexing.
* **Day 10: Consensus Sequence Creator** (`consensus_sequence_creator.py`) - Tracks localized array majority voting to establish democratic gene consensus profiles.
* **Day 11: Amino Acid Weight Calculator** (`amino_acid_weight_calculator.py`) - Leverages direct character iterations to tally total molecular masses.
* **Day 12: Random Sequence Generator** (`random_sequence_generator.py`) - Incorporates Python’s `random` package to construct mock test genomes.
* **Day 13: Hydrophobicity Plotter** (`hydrophobicity_plotter.py`) - Categorizes structural peptide configurations into local environmental positions.
* **Day 14: Final End-to-End Pipeline** (`master_pipeline.py`) - Combines file reading, structural sanitation, transcription, and robust, crash-protected translation with automatic stop-codon termination.
